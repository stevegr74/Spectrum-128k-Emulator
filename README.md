# ZX Spectrum 128K Emulator (C#)

A from-scratch ZX Spectrum 128K emulator written in C# using only standard libraries.

## Features
- Z80 CPU emulation
- 128K memory paging
- Keyboard matrix
- Basic screen rendering
- 128 menu navigation working

## Current Status
- Can boot into 128 menu
- Can enter 48 BASIC / 128 BASIC
- Can run simple BASIC programs
