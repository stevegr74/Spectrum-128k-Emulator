# Security Policy

## Reporting Issues

Security issues can be reported via GitHub Issues.

For this project, there is no separate private reporting channel.

---

## Scope

This project is an emulator and does not handle sensitive user data.

Security concerns are generally limited to:

- Crashes or undefined behaviour
- Unsafe file handling
- Potential execution issues from malformed inputs

---

## Supported Versions

Only the `main` branch is supported.

---

## Response

Issues will be reviewed as time allows. This is a personal project, so response times may vary.
