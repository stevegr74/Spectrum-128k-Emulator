# Code of Conduct

## Our Pledge

This project aims to be a friendly, welcoming place for people interested in emulation, low-level systems, and learning.

Please be respectful, constructive, and patient when interacting with others.

## Expected Behaviour

- Be respectful and considerate
- Assume good intent
- Give constructive feedback
- Stay on topic
- Help keep discussions productive

## Unacceptable Behaviour

- Harassment or personal attacks
- Dismissive or hostile language
- Spam or off-topic discussions
- Repeatedly ignoring project guidelines

## Maintainer Responsibility

The maintainer is responsible for setting direction and ensuring the project remains focused and coherent.

Decisions made by the maintainer are final.

## Scope

This applies to all interactions in:
- Issues
- Pull requests
- Discussions

## Enforcement

If behaviour becomes a problem, the maintainer may:
- Request changes in tone or behaviour
- Close discussions
- Restrict participation if necessary
